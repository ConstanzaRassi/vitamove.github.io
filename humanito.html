Manual de uso
Bienvenido a nuestro humano dinámico!

Este modelo te permite seleccionar distintos grupos musculares para que puedas realizar un entrenamiento 100% localizado.


¿Cómo usarlo?

Debes seleccionar tres grupos musculares y se mostrará en pantalla un tipo de ejercicio que trabajará cada músculo y las repeticiones aconsejadas.

Cada combinación que elijas te permitira realizar ejercicios totalmente diferentes para que trabajes el músculo de distintas maneras
